# Mnemonic Research Context

## Project Focus

Mnemonic is a filesystem-based memory system for Claude Code. Research should focus on:

1. **Competitive landscape** - Other AI memory/context tools
2. **Integration opportunities** - Multi-tool assistant support
3. **Standards adoption** - MIF format, MCP protocol
4. **User needs** - Developer workflows with AI assistants

## Key Competitors/Alternatives

- mem0 (formerly MemGPT)
- Zep
- LangChain Memory modules
- Native tool memories (Cursor, Windsurf)
- Vector database solutions

## Research Priorities

1. MCP ecosystem developments
2. AI coding assistant market trends
3. Memory/context persistence patterns
